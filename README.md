# Admin Dashboard

An interactive Admin Dashboard layout built as part of The Odin Project curriculum. This project serves as a comprehensive practice exercise to master advanced CSS Grid techniques, semantic layout architectures, and complex dashboard user interfaces.

## Live Demo

Check out the live version of the project here: **[Admin Dashboard Live Demo](https://andrea441.github.io/admin-dashboard/)**

## Description

This project is designed to test and build competency in creating complex, asymmetrical web layouts without relying on modern framework grid systems. It features a complete sidebar navigation panel, a multi-tiered header with a dynamic search bar, an interactive greeting interface, and a primary content stream split into projects, announcements, and trending sections.

## Technologies Used

- **HTML5:** Semantic markup structure including header divisions, lists, interactive buttons, and native inline SVG paths for high-density asset scaling.
- **CSS3:** Advanced stylesheet configurations implementing:
  - **CSS Grid & Subgrids:** Utilized for both the global app wrapper (`grid-template-columns: 1fr 3fr; grid-template-rows: 1fr 4fr;`) and individual nested cards.
  - **CSS Custom Variables:** Leveraged for centralized color palette mapping (`--secondary-font-color: gray;`) to manage state typography.

## How to Use

1. **View Online:** Click the **[Live Demo](https://andrea441.github.io/admin-dashboard/)** link to interact with the responsive interface directly inside your web browser.
2. **Run Locally:**
   - Clone this repository to your local computer environment:
     ```bash
     git clone https://github.com/andrea441/admin-dashboard.git
     ```
   - Navigate directly into the cloned workspace directory.
   - Open the entry point `index.html` file using your web browser of choice.
